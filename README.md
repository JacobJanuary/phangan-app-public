# Swyby Mobile App

React + TypeScript + Vite mobile application for discovering and planning local activities.

Part of [Swyby](https://swyby.io) ecosystem — AI-powered activity planner.

## Features

- 🔍 Activity discovery with filters by intent (social, calm, adventure, romantic)
- 📍 Location-based recommendations
- 📅 Day planner with personalized schedule
- 🎫 One-tap booking integration
- 🌙 Dark/light mode support

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 19 |
| Language | TypeScript |
| Build | Vite |
| Styling | Tailwind CSS |
| State | React hooks |
| API | REST (Swyby backend) |

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
