# Project Evolution

## phangan-app-public (Swyby Mobile App)

Public mobile app for Swyby activity discovery and planning.

### Stack
- React 19 + TypeScript + Vite
- Tailwind CSS
- PWA

### Features
- Intent-based activity discovery
- Location-based recommendations
- Personalized day planner
- One-tap booking
- Dark/light mode

This is the user-facing mobile experience of Swyby.
